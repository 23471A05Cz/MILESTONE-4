# 💬 Chatterbox – Real Time Chat Application

Chatterbox is a simple **real-time chat application** built using **FastAPI, WebSockets, HTML, CSS, and JavaScript**.
It allows multiple users to join chat rooms and communicate instantly.

---

## 🚀 Main Features

* 💬 Real-time messaging
* 👥 Multiple chat rooms
* ⌨️ Typing indicator
* 👀 Message seen status
* 😀 Emoji support
* 👍 Message reactions
* ↩️ Reply to messages
* 👥 Online users list
* 🚪 Join and leave chat rooms

---

## 🛠 Technologies Used

* Python
* FastAPI
* WebSockets
* HTML
* CSS
* JavaScript
