
# 💬 Chatterbox - Real-Time Chat Application (Milestone 4)

## 📖 Project Overview

Chatterbox is a real-time web-based chat application built using **FastAPI** and **WebSockets**.
It allows multiple users to communicate instantly through a simple and interactive interface.

Milestone 4 enhances the application with advanced real-time features such as typing indicators, online user tracking, message reactions, and improved performance.

---

## 🚀 Features

* 🔹 Real-time messaging between multiple users
* 🔹 Username-based chat system
* 🔹 Multiple chat room support
* 🔹 Online users list display
* 🔹 Typing indicator for active users
* 🔹 Message reactions using emojis
* 🔹 Reply to specific messages
* 🔹 User join/leave notifications
* 🔹 Instant message broadcasting using WebSockets
* 🔹 Clean and responsive UI
* 🔹 Fast and lightweight backend

---

## 🛠️ Technologies Used

* **FastAPI (Python)** – Backend framework for handling APIs and WebSockets
* **WebSockets** – Enables real-time, bidirectional communication
* **HTML** – Structures the web page
* **CSS** – Styles and improves the UI
* **JavaScript** – Handles client-side logic and WebSocket communication
* **Uvicorn** – ASGI server to run FastAPI
