Chatterbox – Milestone 4 (Real-Time Chat Application)
📖 Project Overview
Chatterbox is a real-time web-based chat application built using FastAPI and WebSockets.
Milestone 4 focuses on implementing live communication features, improved UI, and better user interaction.
🚀 Features
💬 Real-time messaging using WebSockets
👤 Username-based chat system
🟢 Online users tracking
✍️ Typing indicator
📡 Fast and lightweight backend using FastAPI
🌐 Simple and responsive frontend (HTML, CSS, JavaScript)
🛠️ Technologies Used
Backend: FastAPI (Python)
Frontend: HTML, CSS, JavaScript
Protocol: WebSockets
Server: Uvicorn
