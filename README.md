
📖 Project Overview

Chatterbox is a real-time web-based chat application developed using FastAPI and WebSockets.
It enables users to communicate instantly through a simple and interactive interface.
Milestone 4 enhances the project by adding advanced features like typing indicators, online user tracking, and improved real-time performance.



🚀 Features

- Real-time messaging between multiple users
- Username-based chat system
- Online users list display
- Typing indicator for active users
- Instant message broadcasting using WebSockets
- Simple, clean, and responsive user interface
- Lightweight and fast backend processing



🛠️ Technologies Used

- FastAPI (Python): Used to build the backend server and handle WebSocket connections
- WebSockets: Enables real-time, two-way communication between client and server
- HTML: Structures the web page
- CSS: Styles and improves the visual design
- JavaScript: Handles client-side logic and WebSocket communication
- Uvicorn: ASGI server used to run the FastAPI application
